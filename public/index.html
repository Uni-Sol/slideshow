What is GLSL?

WebGL canvas context

  • Short hand for OpenGL Shader Language

  • GLSL written in C like syntax

  • Heavily optimised for GPU

  • Two flavours: Vertex and Fragment Shaders

What is WebGL?

WebGL canvas context

  • Web GL exposes Javascript API

  • Combines with GLSL underneath

  • You can write your own shaders. Not trivial

What is Three.js?

Mr.Doob = Legend

  • Industry standard

  • Abstraction layer over Shaders

  • Reduce massive complexity into simple API

Basic geometry

Show geometry demo

var material = new THREE.MeshBasicMaterial( { color: 0xFF0000 } );
var geometry = new THREE.SphereGeometry(4,20,20);
var mesh = new THREE.Mesh( geometry, material );
mesh.position.set(10, 2, 0);
scene.add( mesh );

Animating scene

Show animation demo

var pos = 10;
function render() {
  pos = pos + 1;
  camera.position.set(pos, 10, 10);
  renderer.render(scene, camera);
  requestAnimationFrame(render);
}
